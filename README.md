## ğŸ› ï¸ Installation & Setup

### Prerequisites
- Node.js (v16+)
- Python (v3.8+)
- Android Studio
- PostgreSQL/MySQL

### Backend
```bash
cd aicte-backend
npm install
npm run prisma:generate
npm run dev
```

### Frontend
```bash
cd aicte-frontend
npm install
npm run dev
```

### AI Models
```bash
cd aicte_models
pip install -r requirements.txt
python main.py
```

### Mobile App
```bash
cd aicte-frontend/android
./gradlew build
```
